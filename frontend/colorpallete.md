# GlassPro — Paleta de colores (sitio web)

## Colores principales (Brand)

| Nombre sugerido | HEX | RGB | Uso recomendado |
|---|---:|---:|---|
| Azul primario | **#0255D1** | 2, 85, 209 | CTAs, enlaces, acentos principales, estados activos |
| Azul secundario | **#1C75BC** | 28, 117, 188 | Botones secundarios, highlights, iconografía |
| Azul profundo | **#0C4C78** | 12, 76, 120 | Headers, fondos oscuros, overlays, elementos de contraste |
| Gris carbón | **#373435** | 55, 52, 53 | Texto principal, títulos, elementos UI neutros |

## Recomendación rápida de uso
- **Fondo general:** blanco o blanco con gradientes/decoracion azul
- **Texto:** `#373435` (y blanco cuando el fondo sea `#0C4C78`).
- **Acciones principales:** `#0255D1`.
- **Acciones secundarias / detalles:** `#1C75BC`.
- **Secciones oscuras / contraste:** `#0C4C78`.
